# Queen's Applied AI Club | QVibe

A modern landing page and website for Queen's Applied AI Club (QVibe), featuring custom pages for About Us, Projects, Events, Our Team, and Get Involved.

## Overview

QVibe is Queen's University's fastest growing AI community, empowering anyone to build with AI tools regardless of technical background.

## Features

- **Hero Section**: Video background with animated title
- **Category Pages**: Custom layouts for each section
- **Team Page**: Hierarchical team structure display
- **Projects & Events**: Interactive cards with modal popups
- **Get Involved**: Contact form for engagement
- **Responsive Design**: Mobile-first approach
- **Modern UI**: Dark mode with gradient accents

## Tech Stack

- React + Vite
- React Router
- GSAP for animations
- CSS with custom properties
